 一、我们如何收集和使用您的个人信息

我们将遵循正当、合法、必要的原则，仅为实现本政策所述目的，收集和使用您在使用本服务过程中主动提供的信息，以及因使用我们的产品或服务而产生的个人信息。若我们拟将您的个人信息用于本政策未载明的其他用途，或基于特定目的将已收集的信息用于其他场景，我们会以合理方式向您充分告知，并在使用前再次获得您的明确同意。如您对本部分内容有任何意见、建议或疑问，可通过文末联系方式与我们联系。


我们致力于为所有用户提供安全、便捷且高效的服务，助力您更轻松、更满意地使用我们的功能。为实现上述目标及本政策所载明的各项服务功能，我们将基于以下目的收集和使用您的信息：


保障应用及相关服务的安全运行，提供更贴合您需求的服务及信息展示

为实现这一目的，我们会收集您使用服务时的设备信息和服务日志信息：


设备信息：包括设备类型、设备名称、设备型号、MAC 地址、唯一设备标识符（如 IMEI/android ID/IDFA/OPENUDID/GUID/SIM 卡 / IMSI）、操作系统及应用程序版本、语言设置、屏幕分辨率、登录 IP 地址、网络设备硬件地址、网络接入方式、服务提供商网络 ID（PLMN）、设备位置相关信息、网络质量数据、移动网络信息（含运营商名称）、产品版本号、浏览器类型等软硬件信息。此外，我们可能会向您使用的设备存储空间写入您访问或使用本应用时产生的数据。


请注意，单独的设备信息、服务日志信息本身无法识别您的自然人身份，属于非个人信息。但若我们将该等信息与其他信息结合后可识别您的特定身份，则在结合使用期间（已通过匿名化、去标识化处理且无法识别特定个人的信息除外），我们会将其视为个人信息，并依据本隐私政策及相关法律规定进行处理和保护。

您理解并同意，我们将通过技术手段对您的上述个人信息进行去标识化、匿名化处理。经处理后的信息将无法识别您的个人身份，在此前提下，我们有权依据法律法规规定，将已去标识化、匿名化的信息用于用户数据库分析及商业化利用。

二、Cookies 和同类技术

为确保应用正常高效运转、为您提供更便捷的访问体验、向您推荐可能感兴趣的内容，我们会在您的设备终端系统中存储名为 Cookies 的小型数据文件。Cookies 通常包含标识符、站点名称及一些字符和号码。Cookies 及同类设备信息标识技术是互联网行业普遍使用的技术，其主要用途包括：保障产品与服务的安全高效运转、排查异常情况、帮助您优化对广告的选择与互动、保护您的数据安全等。


我们不会将 Cookies 用于本隐私政策所述目的之外的任何场景。您有权选择接受或拒绝 Cookies：大多数浏览器会自动接受 Cookies，您可根据自身需求修改浏览器设置以拒绝 Cookies；您也可以清除移动设备上保存的所有 Cookies。但请注意，若您拒绝或清除 Cookies，可能无法完全体验我们提供的部分便捷性及安全性服务功能。


三、我们如何共享、转让或披露您的个人信息


（一）共享

除以下情形外，未经您同意，我们不会与任何第三方共享您的个人信息：


  1. 获得您的明确同意后，我们会与其他方共享您的个人信息；
  2. 基于法律规定、诉讼争议解决需要，或按行政、司法机关依法提出的要求，对外共享您的个人信息；
  3. 为实现本应用服务的基本功能或向您提供所需服务，必须共享您的信息时（具体如下）：
    • 安全保障及数据处理服务商：为保障您的账号及应用运行安全、维护您和我们的合法权益，第三方安全服务商可能收集您的设备信息、网络信息、日志信息等；为持续改进服务，我们可能与数据处理服务商分享您的信息，但未经您同意，不会向其共享可识别您身份的个人信息。
    • 广告及分析服务类授权合作伙伴：为优化应用服务，我们可能向相关合作伙伴提供信息，但除非获得您的许可，不会共享可识别您身份的个人信息（如姓名、电子邮箱等）。
    • 其他合作伙伴：在法律法规允许的范围内，为维护我们及关联方、合作伙伴、您或其他用户的合法权益（如防范欺诈、降低信用风险等），我们可能与其他方共享您的个人信息（但不包括为获利目的出售、出租、共享或以其他方式披露的信息）。
  4. 若您拒绝我们的合作伙伴收集其提供服务所必需的个人信息，可能导致您无法在应用中使用该第三方服务。
  5. 应您的合法需求，协助处理您与他人的纠纷或争议；
  6. 应您的监护人的合法要求提供您的信息；
  7. 学术研究机构基于公共利益开展统计或学术研究，且对外提供研究结果时已对其中的个人信息进行匿名化处理的；
  8. 根据与您签署的用户服务协议（包括电子协议及应用规则）或其他法律文件的约定提供。


（二）转让

我们不会将您的个人信息转让给任何公司、组织或个人，除非符合以下情形：


  1. 因业务发展需要进行合并、收购、资产转让或类似交易时，您的个人信息可能作为交易一部分被转移。我们会要求接收方继续受本隐私政策约束；若其不愿接受，我们将要求其重新获得您的授权同意。
  2. 根据适用法律法规、法律程序要求，或强制性行政、司法要求必须提供的；
  3. 符合与您签署的相关协议（包括书面协议及应用规则）或其他法律文件约定的；
  4. 事先获得您的明确同意或授权的。


（三）公开披露

我们仅在以下情况下，且采取符合业界标准的安全防护措施后，才会披露您的个人信息：


  1. 获得您的明确同意，或基于您的主动选择；
  2. 根据法律法规要求、强制性行政执法或司法要求，必须提供您的个人信息时，我们会要求接收方出具相应法律文件（如传票、调查函），并按要求的类型和方式披露。


（四）第三方 SDK 的使用

为保障客户端稳定运行、实现更多服务功能（如账号登录、安全验证、消息推送、支付、分享、客服及统计分析等），我们的应用中可能嵌入第三方服务提供商的 SDK 或类似应用程序。


第三方服务商可能收集您的相关信息，其信息处理行为不受本隐私政策约束，而受其自身隐私政策规范。为保障您的信息安全，建议您在使用任何第三方 SDK 服务前，仔细查阅其隐私条款。如您发现相关 SDK 存在风险，建议立即终止操作并与我们联系。

（五）共享、转让、公开披露个人信息无需事先授权的例外

根据相关法律法规，以下情形中，共享、转让、公开披露您的个人信息无需事先征得您的同意：


  1. 为履行法定义务或法定职责所必需的；
  2. 依据法律规定，在合理范围内收集、使用个人自行向社会公众公开或其他已合法公开的个人信息；
  3. 为签订或履行与您的合同及其他书面文件所必需的；
  4. 法律法规规定的其他情形。


需特别说明：经匿名化处理的个人信息，因其已无法识别特定个人，其对外共享、转让或公开披露不属于个人信息处理行为，无需另行通知并征得您的同意。


请您注意：


  • 您在使用服务时自愿发布或公开共享的信息（如交易信息、评价中包含个人信息的文字 / 图片 / 视频等），可能涉及您或他人的个人信息（包括敏感信息）。请您谨慎判断是否发布或公开，相关责任由您自行承担。
  • 若您提供的信息包含其他用户的个人信息，需确保已获得合法授权并承担相应责任；若涉及儿童个人信息，需在发布前获得对应儿童监护人的同意。前述情况下，监护人有权通过本政策载明的途径联系我们，要求更正或删除相关儿童个人信息。

  1. 存储地点：我们会按照法律法规规定，将收集的用户个人信息存储于中国境内；若您位于其他国家 / 地区，信息可能存储于您所在的国家 / 地区。

（二）个人信息的保护措施

我们将通过以下手段保护您的个人信息：


  1. 数据安全技术措施：采用符合业界标准的安全防护措施（包括制度规范、安全技术等），防止您的信息被未经授权的访问、使用、披露、修改、损坏或丢失。

  4. 我们会采取合理可行的措施，尽力避免收集无关的个人信息。
  5. 安全事件处置：
    您在通过本应用与第三方进行交易时，可能需向交易对方披露个人信息（如联系方式、地址等），请您妥善保护自身信息，仅在必要时提供。
